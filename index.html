What is a JOIN?

A JOIN in SQL is used to combine rows from two or more tables based on a related column.

INNER JOIN

Returns rows that have matching values in both tables.

SELECT Customers.Name, Orders.Product 
FROM Customers 
INNER JOIN Orders
ON Customers.CustomerID = Orders.CustomerID;

LEFT JOIN

Returns all rows from the left table, and the matched rows from the right table. Unmatched right table fields will be NULL.

RIGHT JOIN

Returns all rows from the right table and matched rows from the left. Unmatched left table fields will be NULL.

FULL JOIN

Combines results of both LEFT and RIGHT joins. Unmatched rows from both sides are included.

CROSS JOIN

Returns the Cartesian product ‚Äì all combinations of rows from the two tables.

No overlapping sets ‚Äì every row from table A with every row from table B.

SELF JOIN

Used when a table is joined with itself to compare rows within the same table.
